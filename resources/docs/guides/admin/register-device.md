# How to Register a Device

## Prerequisites
- Device (tablet or relay) ready for setup
- Network access configured
- Branch and table information prepared

## Steps
1. Navigate to "Devices" from the sidebar
2. Click "Add Device" button
3. Fill in device information:
   - Device name (descriptive identifier)
   - Device type (Tablet or Relay)
   - Branch assignment
   - Table assignment (for tablets)
4. Click "Generate Token"
5. Copy the generated token
6. On the physical device, enter the token in the device app
7. Verify connection status shows "Connected"

## Expected Outcome
The device will appear in the devices list with a green "Active" status and begin receiving updates in real-time.

## Troubleshooting
- **Token not working:** Ensure token was copied completely without extra spaces
- **Device offline:** Check network connectivity on the device
