# How to Manage Orders

## Viewing Orders
1. Click "Orders" in the sidebar to see all active orders
2. Use filters to narrow results:
   - By status (Pending, Processing, Completed)
   - By branch
   - By date range
3. Click on any order to view full details

## Completing an Order
1. Open the order details
2. Review all items and their status
3. Click "Mark as Complete"
4. Confirm the action in the dialog

## Printing an Order
1. Open the order details
2. Click the "Print" button
3. Select the target printer (if multiple available)
4. The order will be sent to the printer device

## Voiding an Order
1. Open the order details
2. Click "Void Order" button
3. Enter a reason for voiding (optional)
4. Confirm the action

## Expected Outcome
- Completed orders move to the "Completed" tab
- Printed orders show a "Printed" timestamp
- Voided orders are marked clearly and excluded from reports
