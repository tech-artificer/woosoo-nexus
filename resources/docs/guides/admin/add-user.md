# How to Add a New User

## Prerequisites
- Admin or User Manager role required
- Branch assignment prepared

## Steps
1. Click on "Users" in the sidebar navigation
2. Click the "Add User" button in the top right corner
3. Fill in the user details:
   - Username (required, unique)
   - Email address (required, valid format)
   - Password (minimum 8 characters)
   - Select branch assignment
4. Assign role(s) from the available list
5. Click "Create User" to save

## Expected Outcome
The new user will appear in the users list and can log in immediately with the provided credentials.

## Tips
- Use strong passwords combining letters, numbers, and symbols
- Assign only necessary permissions following the principle of least privilege
