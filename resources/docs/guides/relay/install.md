# How to Install the Relay App

## Download the App
1. On your Android device, open a web browser
2. Navigate to the relay app download URL (provided by admin)
3. Download the APK file
4. Note the download location (usually Downloads folder)

## Enable Installation from Unknown Sources
1. Open Android **Settings**
2. Go to **Security** or **Privacy**
3. Find **Install Unknown Apps** or **Unknown Sources**
4. Select your browser (Chrome, Firefox, etc.)
5. Toggle **Allow from this source** ON

## Install the APK
1. Open your file manager app
2. Navigate to Downloads folder
3. Tap on the relay app APK file
4. Tap **Install** when prompted
5. Wait for installation to complete
6. Tap **Open** or find the app in your app drawer

## First Launch Setup
1. Grant Bluetooth permission when prompted
2. Grant Storage permission when prompted
3. Grant Network permission when prompted
4. Enter the device token from the admin panel
5. Tap **Connect**
6. Wait for connection confirmation

## Expected Outcome
App shows "Connected" status and is ready to receive print jobs from the server.

## Troubleshooting
- **Installation blocked:** Verify unknown sources setting is enabled
- **App won't open:** Try restarting device
- **Permissions denied:** Manually grant permissions in Android Settings > Apps
