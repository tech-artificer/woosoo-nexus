# How to Navigate the Tablet Menu

The tablet menu is designed to be intuitive for guests of all ages. This guide covers the interface layout and navigation controls.

## Category Tabs

At the top of the menu screen, horizontal tabs separate items by category (e.g., Appetizers, Main Course, Grilled, Drinks, Desserts).

- **Tap a tab** to instantly switch to that category.
- Categories and their order are configured by the admin under Tablet Categories.
- If there are more categories than fit on screen, swipe the tab bar left or right to reveal more.

---

## Menu Item Cards

Each menu item is shown as a card with:
- A **photo** of the dish (configured in the admin)
- **Item name**
- **Price**
- A **"Sold Out"** overlay if the item is currently unavailable

Items that are sold out appear grayed out and cannot be tapped to add to the cart.

---

## Viewing Item Details

1. **Tap any item card** that is not sold out.
2. A detail modal opens with:
   - Larger photo
   - Full item description
   - Price
   - **Modifier options** (e.g., spice level — Mild / Medium / Hot; extras — Extra rice, Extra sauce)
3. Select your preferred modifiers by tapping the option buttons.
4. Adjust the quantity using the **+** and **−** buttons (minimum 1).
5. Tap **Add to Cart** to add the item, or tap the **X** / anywhere outside the modal to close without adding.

---

## The Cart

The cart icon is visible at all times (top-right corner) and shows a badge with the current number of items.

- Tap the cart icon to open the **Cart Screen**.
- From the Cart Screen you can:
  - Review all added items
  - Adjust quantities (tap + or −)
  - Remove items (tap the trash icon)
  - Add a note for the kitchen
  - Proceed to submit the order

---

## Call Staff Button

The bell icon in the top area opens a **Service Request** panel. Guests can choose the type of assistance:
- General assistance
- Refill request
- Bill request
- Other

A notification appears immediately on the admin dashboard under Service Requests.

---

## Return to Welcome Screen

If the tablet needs to be reset to the Welcome screen (session not yet started), tap **Home** or look for the back button at the top left. Only staff can reset an active session via the admin panel.

---

## Tips for Smooth Use

| Tip | Details |
|-----|---------|
| **Scrolling** | Swipe up and down within the item grid to see more items |
| **Sold out items** | Are visible but grayed — the restaurant keeps them visible so guests know what's on the menu |
| **Modifiers** | Some items require a modifier selection before "Add to Cart" becomes active |
| **Multiple submissions** | Guests can submit multiple orders during a session — the kitchen gets each batch separately |
