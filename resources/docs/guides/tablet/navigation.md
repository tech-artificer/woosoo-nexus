# How to Navigate the Menu

## Interface Overview
- **Category Tabs:** Horizontal tabs at the top for filtering by course type
- **Item Cards:** Grid of menu items with images, names, and prices
- **Cart Button:** Fixed button showing cart item count
- **Home Button:** Returns to welcome screen

## Browsing Categories
1. Tap on category tabs (Appetizers, Main Course, Desserts, etc.)
2. Category content loads instantly
3. Scroll vertically to see all items in the category

## Viewing Item Details
1. Tap on any item card
2. Modal opens with:
   - Large item image
   - Full description
   - Available modifiers (spice level, extras, etc.)
   - Price
3. Tap outside modal or X button to close

## Using Search (if enabled)
1. Tap the search icon
2. Type item name or keyword
3. Results filter in real-time
4. Tap item to view details

## Tips
- **Swipe gestures:** Swipe left/right on category tabs for quick navigation
- **Image quality:** Tap item image for larger view
- **Sold out items:** Appear grayed with "Sold Out" overlay
