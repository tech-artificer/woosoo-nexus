# How to Access Settings

## Security Note
Settings are PIN-protected to prevent unauthorized access by guests.

## Accessing Settings
1. Tap the settings icon (gear icon, usually in corner)
2. Enter the 4-digit PIN when prompted
3. Settings screen opens

## Available Settings
- **Device Information:** View device token, connection status
- **Display Options:** Brightness, timeout settings
- **Network:** WiFi configuration
- **Maintenance Mode:** Temporarily disable ordering
- **Clear Cache:** Refresh menu data
- **Logout Device:** Disconnect from server

## Exiting Settings
1. Tap "Done" or "Close" button
2. Settings auto-lock after 2 minutes of inactivity
3. Device returns to guest interface

## Security Best Practices
- Keep PIN confidential
- Change PIN periodically
- Log out when performing maintenance
