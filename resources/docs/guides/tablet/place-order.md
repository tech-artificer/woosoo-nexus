# How to Place an Order (Guest Flow)

## Complete Ordering Process

### Step 1: Welcome Screen
1. Tap anywhere on the welcome screen to begin
2. System loads the menu catalog

### Step 2: Enter Guest Count
1. Enter the number of guests at your table
2. Tap "Continue"

### Step 3: Select Package (Optional)
1. Browse available package deals
2. Tap on a package to view details
3. Select package or skip to browse individual items

### Step 4: Browse Menu
1. Use category tabs to navigate (Appetizers, Main Course, Drinks, etc.)
2. Scroll to view all items in each category
3. Tap on an item card to view details

### Step 5: Add Items to Cart
1. Select any modifiers or options
2. Adjust quantity using +/- buttons
3. Tap "Add to Cart"
4. Continue browsing or proceed to cart

### Step 6: Review Cart
1. Tap the cart icon (shows item count badge)
2. Review all items in your order
3. Adjust quantities or remove items if needed
4. Add special instructions if necessary

### Step 7: Submit Order
1. Tap "Submit Order" button
2. Review order summary
3. Confirm submission

### Step 8: Order Confirmation
1. Order number appears on screen
2. Estimated preparation time shown
3. Return to home screen or browse more items

## Expected Outcome
Your order is sent immediately to the kitchen and printer. Kitchen staff can view and prepare your order.
