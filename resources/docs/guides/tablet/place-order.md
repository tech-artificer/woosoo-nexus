# How to Place an Order — Guest Flow

This guide covers the complete guest experience from the welcome screen to order confirmation. All steps happen on the tablet mounted at the guest's table.

---

## Step 1 — Welcome Screen

1. The tablet displays a **Welcome** screen when no session is active.
2. **Tap anywhere** on the screen to begin.
3. The app moves to the guest count screen.

---

## Step 2 — Enter Guest Count

1. A number pad appears asking: **"How many guests are dining today?"**
2. Tap the number of guests at your table.
3. Tap **Continue**.

> This count is recorded with the order and appears in guest reports in the admin.

---

## Step 3 — Select a Package (Optional)

If the restaurant offers dining packages (e.g., eat-all-you-can, set meals):

1. A package selection screen appears showing available packages.
2. Tap on a package card to read the description and price.
3. Tap **Select Package** to choose it, or tap **Order à la carte** to skip and browse the full menu freely.

> Once a package is selected, the menu will show only the items included in that package.

---

## Step 4 — Browse the Menu

1. The menu screen shows **category tabs** across the top (e.g., Appetizers, Main Course, Drinks, Desserts).
2. Tap a category tab to switch between sections.
3. Scroll up and down to see all items.
4. Items marked **"Sold Out"** are shown in gray — they cannot be added to the cart.

---

## Step 5 — View Item Details & Add to Cart

1. **Tap on any item card** to open the item detail view.
2. The detail view shows:
   - Item name and description
   - Price
   - Available modifiers (e.g., spice level, extras, cooking style)
3. Select any desired modifiers using the option buttons.
4. Adjust the quantity using the **+** and **−** buttons.
5. Tap **Add to Cart**.
6. The cart icon (bottom of screen) updates to show the total number of items.

---

## Step 6 — Review the Cart

1. When ready to order, tap the **Cart** icon at the bottom of the screen.
2. The cart screen shows all added items with:
   - Item names and selected modifiers
   - Individual quantities (can be adjusted here)
   - Per-item prices
   - Total price
3. Remove any item by tapping the **trash icon** next to it.
4. Tap **−** or **+** to adjust quantity.
5. Add any special instructions in the **Notes** field (optional).

---

## Step 7 — Submit the Order

1. Tap the **Submit Order** button.
2. A confirmation screen appears — review the final order summary.
3. Tap **Confirm** to send the order.

**Expected result:** The order is sent immediately to:
- The admin dashboard (appears in Live Orders)
- The print relay device (sent to the kitchen printer)

---

## Step 8 — Order Confirmation Screen

After submitting:
1. The screen displays your **Order Number** (e.g., `#0042`).
2. A "Your order is being prepared" message is shown.
3. After a short delay, the screen transitions back to the menu to allow additional orders.

---

## During the Dining Session

### Placing Additional Orders (Refills / Add-ons)

While a session is active, guests can continue ordering:
1. Browse the menu and add more items to the cart.
2. Submit additional orders — these are added to the same session.

### Requesting Service

To call a staff member:
1. Tap the **Call Staff** or **Service** button (bell icon in the corner).
2. Select the type of request (e.g., Assistance, Refill, Bill).
3. Tap **Send**.
4. Staff can see the request in the admin panel under **Service Requests**.

---

## Session End

The session ends when:
- Staff marks the order as **Completed** in the admin.
- The tablet returns to the **Welcome** screen for the next table of guests.

---

## What to Do If Something Goes Wrong

| Problem | What the Guest Should Do |
|---------|-------------------------|
| Cart shows old/wrong items | Tap the cart icon and remove incorrect items |
| App froze while ordering | Wait 10 seconds — the app will recover. If not, call a staff member. |
| "Unable to submit order" error | Call a staff member — the server may be temporarily unreachable |
| Already submitted an order but want to cancel | Call a staff member to void the order in the admin panel |
