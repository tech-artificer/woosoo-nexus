# Tablet Ordering System Requirements

## Hardware Requirements
- Tablet device (iPad, Android, or Windows tablet)
- Minimum 10" screen size recommended
- Touch screen support
- Stable stand or mount for table placement

## Software Requirements
- Modern web browser with PWA support
- Minimum OS versions:
  - iOS/iPadOS 14+
  - Android 10+
  - Windows 10+

## Network Requirements
- WiFi connection (5 GHz recommended for better performance)
- Minimum 5 Mbps download speed
- WebSocket support (for real-time updates)
- Access to admin server URL

## Deployment Checklist
1. Connect tablet to WiFi network
2. Open browser and navigate to tablet app URL
3. Install as PWA (Add to Home Screen)
4. Enter device token from admin panel
5. Verify connection status
6. Test order placement
7. Mount tablet in kiosk stand on table
