## What each step fixes

### Step 1 — Sidebar & topbar shell
- Active nav items: white bg → amber-tinted (`#2a1e0c`) with amber text (`#F6B56D`) and 2px left bar
- Logo area: glassmorphism pill → flat logo row with divider
- Page header: floating frosted-glass pill → flat flush 52px topbar

### Step 2 — Layout & navigation
- Topbar right: adds sun/moon dark mode toggle (persists to `localStorage`)
- Content wrapper: `rounded-[30px]` → `rounded-[26px]`; `backdrop-blur-xl` → `backdrop-blur-sm`

### Step 3 — UI primitives
- **Badge**: new `warning` (amber) variant; dark-mode fixes for `success`, `active`, `accent`
- **Button**: `rounded-xl` → `rounded-lg` (aligns with `--radius: 0.5rem`)
- **Card**: `border-white/65` → `border-black/8` (visible in light mode); softer shadow; lighter blur
- **TabsList**: warm bordered container; `h-9` → `h-10`
- **TabsTrigger**: white-card active state with border and shadow

### Step 4 — Page-by-page
- **Dashboard**: icon containers `rounded-2xl` → `rounded-xl`; card borders unified
- **Orders**: Echo status pill and Live Orders badge use brand tokens instead of raw Tailwind colors
- **Devices**: removed double-wrapped glass-card layout; inline 2-col stats replace 4-col `StatsCards`
