Change 05 / 05
Active sidebar left-bar indicator
● Medium impact
The brand mockup shows a 2px amber bar on the left edge of the active nav item. This can't be done purely with Tailwind utility classes on the button — add it as a CSS rule in app.css.
resources/css/app.css
/* Add at the bottom of app.css */

/* Sidebar active item — left accent bar */
[data-sidebar="menu-button"][data-active="true"] {
    position: relative;
}
[data-sidebar="menu-button"][data-active="true"]::before {
    content: '';
    position: absolute;
    left: -8px;   /* sits outside the button, in the sidebar padding */
    top: 6px;
    bottom: 6px;
    width: 2px;
    background: #F6B56D;  /* --color-woosoo-accent */
    border-radius: 0 2px 2px 0;
}

/* Tighten sidebar nav item border-radius (optional — brand uses 6px) */
[data-sidebar="menu-button"] {
    border-radius: 6px !important;
}
