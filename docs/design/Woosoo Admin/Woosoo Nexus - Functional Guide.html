How to use this document

  1. Read the screen. Each section opens with the page's purpose and an annotated screenshot.
  2. Follow "How it works". Numbered steps walk the operator through every function.
  3. Run the verification. The green Verify line under each step tells you exactly what to click and what you should observe — use it to confirm the build matches this spec.
  4. Note the gaps. The amber/red box lists controls that do nothing yet.
  5. Work the checklist. The final section is a tracked, tickable backlog of every missing feature — your progress is saved in this browser.
